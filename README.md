# Ping Pong Game (Python + Pygame)

This project implements a classic two-player Ping-Pong game in Python using
Pygame. It includes paddle controls, ball physics with collisions, and score
tracking.

## Requirements

- Python 3.10+ (Python 3.12 recommended)

Install dependencies:

```bash
python3 -m pip install -r requirements.txt
```

## Run the Game

```bash
python3 ping_pong.py
```

## Controls

- Left paddle: `W` / `S`
- Right paddle: `Up` / `Down`
- Quit: `Esc` or window close button
